# 基于 go-cqhttp 和 Solidjs 的 CQ 客户端

> 该客户端只会实现一些基本的功能，不会实现过于复杂的功能

## 待完善功能

- [ ] 聊天窗口的昵称显示
- [ ] 图片和表情显示
- [ ] 图片发送
- [ ] 文件发送

## dev 方式

- 克隆本仓库，并使用 pnpm 安装依赖
- 在 `go-cq` 目录下存放 [Mrs4s/go-cqhttp](https://github.com/Mrs4s/go-cqhttp) 的可执行文件，将 `config.yml` 中 `report-self-message` 字段设置为 `true`
- `pnpm run dev`
